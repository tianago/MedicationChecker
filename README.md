# MedicationChecker 💊

おじいちゃんのための薬チェック管理アプリ。シンプルで使いやすく、飲み忘れの通知機能付き。

## 特徴

- ✅ **シンプルなデザイン** - 白と灰色のみのシンプルでスタイリッシュなUI
- ⏰ **飲む時間の記録** - 薬を飲んだ時間を自動で記録
- 🔔 **飲み忘れ通知** - 飲む時間を超過したら自動で通知
- 🤖 **GPTに相談** - 健康に関する質問をGPTに相談可能
- 💾 **データ保存** - ブラウザのローカルストレージに保存（永久的に使用可能）
- 📱 **スマートフォン対応** - PWAとして使用可能。ホーム画面に追加できます
- 🌐 **オフライン対応** - 一度起動すればオフラインでも使用可能

## インストール方法

### 方法1: ブラウザで開く
1. このリポジトリをクローン、またはファイルをダウンロード
2. ブラウザで `index.html` を開く

### 方法2: スマートフォンにインストール
1. スマートフォンのブラウザでこのページにアクセス
2. ブラウザのメニューから「ホーム画面に追加」を選択
3. アプリがホーム画面に追加されます

## 使い方

### 1. APIキーを設定
1. 「設定」タブを開く
2. OpenAI APIキーを入力
3. 「保存」を押す

### 2. 薬を追加
1. 「設定」タブで薬の情報を入力
   - 薬の名前（例：血圧の薬）
   - 飲む時間（時刻を選択）
2. 「追加」を押す

### 3. 毎日の確認
1. 「今日の薬」タブで確認
2. 薬を飲んだら「飲んだ」ボタンを押す
3. 飲む時間を超過した場合は自動で通知が表示されます

### 4. GPTに相談
1. 「相談」タブを開く
2. 質問を入力（例：「この薬の副作用について教えてください」）
3. 「相談する」を押す
4. GPTからのアドバイスが表示されます

## 通知について

**初回通知**: 指定した時間に到達すると通知が表示されます

**追加通知**: 指定時間を超過してもまだ「飲んだ」ボタンを押していない場合、アプリを開いた時に再度通知が表示されます

## データについて

- すべてのデータはお使いのデバイスのブラウザに保存されます
- クラウドには保存されないため、プライバシーが守られます
- データをバックアップしたい場合は、「設定」タブから「データをエクスポート」を使用できます

## トラブルシューティング

### 通知が表示されない
- ブラウザの通知許可設定を確認してください
- プライベートモードではService Workerが機能しない場合があります

### GPTに相談できない
- APIキーが正しく入力されているか確認してください
- OpenAI APIにアクセス権があるか確認してください
- ネットワーク接続を確認してください

### データが消えた
- ブラウザのキャッシュをクリアするとデータが削除されます
- 事前に「データをエクスポート」でバックアップを取ることをお勧めします
